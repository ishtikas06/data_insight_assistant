A GenAI-powered chatbot that enables users to query organizational data—such as employee salaries, departments, and other HR/operational details—using natural language. With this assistant, users can ask questions like:

* "Who earns the highest salary?"

* "Which department has the most employees?"

* "What is the average salary in the Finance department?"

The assistant processes business data and returns accurate, context-aware insights without requiring SQL or technical knowledge.
